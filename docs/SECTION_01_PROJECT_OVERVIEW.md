# 1. 프로젝트 개요

## 1.1 서비스 소개

### NALO - 날로 먹는 아이디어 기획

**NALO**는 AI 기반의 아이디어 리서치 & 마인드맵 브레인스토밍 플랫폼입니다.

> **슬로건**: "날로 먹는 아이디어 기획 - AI 리서치와 마인드맵으로 아이디어를 구체화하세요"

NALO는 단순한 아이디어 생성 도구를 넘어서, **체계적인 리서치 → 시각적 브레인스토밍 → 전문적인 기획서 생성**까지 전 과정을 자동화한 통합 플랫폼입니다.

### 서비스의 탄생 배경

많은 개발자, 기획자, 창업자들이 좋은 아이디어를 떠올리지만, 이를 구체화하고 체계적인 기획서로 만드는 과정에서 어려움을 겪습니다:

- **리서치의 어려움**: 시장 조사, 학술 논문 검색, 경쟁사 분석 등에 많은 시간 소요
- **구조화의 어려움**: 막연한 아이디어를 체계적으로 정리하기 힘듦
- **전문성의 부족**: 투자자나 팀원에게 제시할 전문적인 기획서 작성 능력 부족

NALO는 이러한 문제를 **AI 기술**과 **자동화된 워크플로우**로 해결합니다.

## 1.2 핵심 가치 제안

### 🎯 3가지 핵심 가치

#### 1. **AI 기반 전문 리서치**
- Wikipedia, 학술 논문(OpenAlex), 웹 리서치(Perplexity AI)를 자동 통합
- 수작업으로 몇 시간 걸리던 리서치를 **수 분 내**로 완료
- 관련성 점수 기반 자동 필터링으로 **질 높은 정보만 선별**

#### 2. **시각적 마인드맵 브레인스토밍**
- 복잡한 아이디어를 **시각적으로 구조화**
- AI 채팅을 통한 **실시간 아이디어 확장**
- 웹 검색 기반 **스마트 확장** 기능으로 최신 트렌드 반영

#### 3. **전문적인 기획서 자동 생성**
- 리서치 데이터 또는 마인드맵을 기반으로 **체계적인 기획서 자동 생성**
- 시장 분석, 기술 스택, 예산 산정, 리스크 분석까지 포함
- PDF 다운로드 및 공유 기능으로 **즉시 활용 가능**

## 1.3 타겟 사용자

### 🎓 개발자 (Developer)
- **해커톤, 사이드 프로젝트** 아이디어 구체화
- 기술 스택 선정 및 아키텍처 설계 지원
- 빠른 프로토타이핑을 위한 체계적 기획

### 📊 기획자 (Product Manager / Planner)
- **신규 서비스 기획** 시 시장 조사 자동화
- 경쟁사 분석 및 차별화 전략 수립
- 프레젠테이션용 전문 기획서 생성

### 🚀 창업자 (Entrepreneur / Startup Founder)
- **스타트업 아이템 검증** 및 구체화
- 투자 유치를 위한 IR 자료 기반 작성
- MVP 개발 전 체계적인 사업 계획 수립

### 🎨 학생 및 연구자 (Student / Researcher)
- 졸업 프로젝트, 연구 과제 아이디어 발굴
- 학술 논문 기반 선행 연구 조사 자동화
- 아이디어 시각화 및 구조화

## 1.4 서비스의 차별점

### 기존 서비스와의 비교

| 기능 | NALO | ChatGPT | Notion AI | Miro/FigJam |
|------|------|---------|-----------|-------------|
| **AI 아이디어 생성** | ✅ 전문 프롬프트 | ✅ 범용 | ✅ 문서 중심 | ❌ |
| **자동 리서치** | ✅ 3개 소스 통합 | ⚠️ 단일 응답 | ❌ | ❌ |
| **마인드맵 편집** | ✅ AI 통합 | ❌ | ❌ | ✅ 수동 |
| **기획서 자동 생성** | ✅ 체계적 템플릿 | ⚠️ 프롬프트 필요 | ⚠️ 문서 보조 | ❌ |
| **완전 무료** | ✅ | ❌ (유료) | ❌ (유료) | ⚠️ (제한적) |
| **통합 워크플로우** | ✅ 원스톱 | ❌ | ❌ | ❌ |

### NALO만의 강점

1. **End-to-End 워크플로우**: 아이디어 발굴 → 리서치 → 브레인스토밍 → 기획서까지 한 곳에서
2. **다중 소스 리서치**: Wikipedia, OpenAlex, Perplexity AI 동시 검색 및 통합
3. **AI 기반 마인드맵**: 단순 시각화가 아닌, AI가 함께 생각하고 확장하는 브레인스토밍
4. **전문 기획서 템플릿**: 개발자/기획자가 실제로 사용할 수 있는 실용적인 포맷
5. **완전 무료 서비스**: API 키만 있으면 무제한 사용 가능

- 🎯 AI 에이전트 기반 프로젝트 관리 도구로 확장

## 1.6 서비스 사용 예시

### Case 1: 사이드 프로젝트 기획
```
1. 사용자: "반려동물 건강 관리 앱" 프롬프트 입력
2. NALO: 주제 탐색으로 "펫테크", "헬스케어 IoT", "AI 진단" 등 확장
3. 사용자: 관심 주제 선택 후 리서치 실행
4. NALO: Wikipedia(펫테크 시장), OpenAlex(AI 질병 진단 논문), Perplexity(최신 트렌드) 통합
5. NALO: 리서치 기반으로 3개 아이디어 자동 생성
6. 사용자: 마음에 드는 아이디어로 상세 기획서 생성
7. 결과: 시장 분석, 기술 스택, 예산, 일정이 포함된 전문 기획서 완성
```

### Case 2: 마인드맵 브레인스토밍
```
1. 사용자: "환경 보호 소셜 미디어" 프롬프트 입력
2. NALO: 자동으로 마인드맵 생성 (중심 노드 + 5-7개 하위 아이디어)
3. 사용자: AI 채팅으로 "게임화 요소 추가해줘" 요청
4. NALO: 마인드맵에 "포인트 시스템", "랭킹", "챌린지" 노드 자동 추가
5. 사용자: "탄소 절감" 노드를 스마트 확장 (웹 검색)
6. NALO: 최신 웹 데이터 기반으로 하위 아이디어 확장
7. 사용자: 완성된 마인드맵으로 기획서 생성
8. 결과: 구조화된 아이디어가 반영된 체계적 기획서 완성
```

## 1.7 기술적 비전

NALO는 단순한 도구가 아닌, **AI와 인간의 협업을 통한 창의적 사고의 증폭기**를 목표로 합니다.

### 핵심 철학
- **AI는 조력자, 인간은 의사결정자**: AI가 방대한 정보를 처리하고 제안하지만, 최종 선택은 사용자
- **속도와 품질의 균형**: 빠른 생성과 동시에 전문적 품질 유지
- **개방성과 확장성**: 오픈 소스 정신에 기반한 투명한 개발, 커뮤니티 기여 환영

### 기술적 원칙
1. **서버리스 우선**: Vercel + Next.js API Routes로 확장성과 비용 효율성 확보
2. **AI 모델 중립성**: OpenAI, Perplexity 등 다양한 AI 모델 통합 가능한 아키텍처
3. **사용자 프라이버시**: 최소한의 데이터만 저장, 투명한 데이터 정책
4. **오픈 소스 친화적**: 주요 로직과 템플릿 공개로 커뮤니티 기여 유도
