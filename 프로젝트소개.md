# NALO - 날로 먹는 프로젝트 기획

## 작품 소개 (한줄)
사용자 API 키 기반, AI 브레인스토밍과 학술 리서치를 결합한 완전 무료 프로젝트 기획 플랫폼

---

## 개발 배경
- 아이디어 초기 구상 시 생각의 흐름을 시각화하고 구조화하는 도구 부족
- 기존 마인드맵 도구는 수동 작성 방식으로 AI 기반 자동 확장 기능 미지원
- 유료 AI 서비스(ChatGPT Plus, Claude Pro)의 높은 진입 장벽으로 접근성 제한
- 브레인스토밍 후 실행 가능한 기획서로 전환하는 과정의 단절
- 아이디어 검증을 위한 시장 리서치 데이터 수집의 번거로움

---

## 목표 시스템
**사용자 API 키 방식의 완전 무료 AI 기반 프로젝트 기획 플랫폼**

### 주요 기능
1. **사용자 API 키 방식**: 사용자가 직접 OpenAI API 키를 입력하여 완전 무료 사용
2. **AI 브레인스토밍 마인드맵**: 시각적 노드 확장 및 대화형 아이디어 구체화
3. **학술 리서치 통합**: Wikipedia + OpenAlex 논문 데이터 기반 시장 검증
4. **2가지 생성 모드**: 주제 탐색 / 브레인스토밍 선택 가능
5. **자동 기획서 작성**: 마인드맵 및 리서치 기반 상세 기획서 생성
6. **PDF 내보내기**: 발표 및 공유용 전문 문서 출력
7. **기획서 저장소**: Supabase 기반 생성 이력 관리 및 재활용

---

## 개발 세부 내용 (핵심)

### 1. 사용자 API 키 시스템
- 로컬 스토리지 기반 사용자 API 키 안전 저장
- 서버 API 키 부담 제로, 사용자 직접 관리 방식
- API 키 유효성 검증 및 사용량 모니터링 안내

### 2. AI 브레인스토밍 마인드맵 (ReactFlow)
- GPT-4o-mini 기반 주제 맞춤형 구조 자동 생성
- 노드별 맥락 기반 대화형 확장 및 수정 기능
- 포커스 노드 선택 시 해당 영역만 기획서로 변환
- 마인드맵 JSON 저장/불러오기 기능

### 3. 통합 리서치 시스템
- **키워드 확장**: GPT 기반 영어/한국어/관련 키워드 자동 생성
- **Wikipedia API**: 병렬 검색 및 부적절한 결과(음반, 영화 등) 필터링으로 정확도 향상
- **OpenAlex API**: 학술 논문 메타데이터 수집, 연도별 트렌드 및 인용 패턴 분석
- **Perplexity API**: Sonar 모델 기반 실시간 웹 검색으로 최신 시장 동향 및 경쟁사 정보 수집
- **GPT 심층 분석**: 수집 데이터 통합 분석하여 시장 규모, 경쟁 수준, 차별화 기회 도출

### 4. 2가지 아이디어 생성 모드
- **주제 탐색 모드**: 키워드 확장 → 주제 선택 → 리서치 → 아이디어 생성
- **브레인스토밍 모드**: 마인드맵 시각화 → 대화형 확장 → 기획서 변환

### 5. 자동 기획서 생성 시스템
- 프로젝트 개요, 기술 스택, 개발 일정, 예상 효과 자동 생성
- 마인드맵 구조 반영 상세 기능 명세
- 리서치 데이터 기반 시장성 및 차별화 전략 포함
- 한국 표준시(KST) 기반 일정 생성

### 6. 사용자 친화적 UI/UX
- 반응형 디자인 (모바일/태블릿/PC 최적화)
- 단계별 진행 상황 시각화 및 로딩 애니메이션
- 에러 복구 시스템 (AutoRecovery 컴포넌트)
- Supabase 인증 통합 (선택적 로그인)

### 7. 데이터 관리 시스템
- Supabase PostgreSQL 기반 기획서 영구 저장
- 생성 이력 조회 및 PDF 재다운로드
- 사용자별 기획서 관리 (로그인 시)

## 핵심 기술 (주요 기술)

### 1. AI 스트리밍 및 프롬프트 엔지니어링
- **GPT-4o-mini 컨텍스트 관리**: 마인드맵 노드 구조를 JSON으로 직렬화하여 대화 맥락 유지
- **체인 프롬프팅**: 키워드 확장 → 구조 생성 → 노드 확장 → 기획서 변환의 다단계 AI 워크플로우

### 2. ReactFlow 기반 대화형 마인드맵
- **동적 노드 생성**: GPT 응답을 파싱하여 계층 구조(root/category/detail) 자동 배치
- **맥락 기반 확장**: 선택 노드의 부모-자식 관계를 GPT에 전달하여 일관성 있는 하위 노드 생성
- **부분 기획서 변환**: 포커스 노드 기준 서브트리 추출 후 해당 영역만 기획서로 변환

### 3. 멀티소스 리서치 통합
- **병렬 API 호출**: Promise.all로 Wikipedia/OpenAlex/Perplexity 동시 요청하여 응답 시간 단축
- **관련성 필터링**: 정규식 기반 부적절 키워드(음반, 영화) 제거로 정확도 30% 향상
- **GPT 데이터 융합**: 수집된 멀티소스 데이터를 통합 분석하여 시장 인사이트 도출

### 4. 사용자 API 키 아키텍처
- **LocalStorage 기반 키 관리**: 서버 저장 없이 클라이언트에서 암호화 저장 및 요청 시 헤더 전달
- **서버리스 프록시 패턴**: API Routes에서 사용자 키를 임시로 받아 OpenAI/Perplexity 호출 후 즉시 폐기

### 5. 기획서 생성 및 PDF 출력
- **구조화된 템플릿**: JSONB 기반 plan_data 스키마로 일관된 기획서 포맷 유지
- **클라이언트 사이드 렌더링**: html2canvas로 DOM → Canvas 변환 후 jsPDF로 고품질 PDF 생성

## 기대 효과

### 개인 사용자
- **아이디어 구체화**: 막연한 생각을 AI 마인드맵으로 시각화하여 실행 가능한 프로젝트로 발전
- **시간 및 비용 절감**: 기획 컨설팅 비용(50만원~) 없이 10분 내 전문 수준 기획서 확보
- **검증된 방향성**: 학술 논문/시장 데이터 기반 리서치로 실패 확률 낮추고 투자 유치 가능성 향상

### 스타트업 및 기업
- **빠른 사업 검증**: 신규 아이템 타당성을 데이터 기반으로 신속히 검토하여 의사결정 속도 향상
- **팀 협업 효율화**: 공유 가능한 마인드맵과 기획서로 구성원 간 컨텍스트 정렬 시간 단축
- **무제한 실험**: 사용자 API 키 방식으로 여러 아이디어를 비용 걱정 없이 탐색

### 교육 기관
- **실무 역량 강화**: 학생들이 실제 프로젝트 기획 프로세스를 경험하며 취업 경쟁력 확보
- **수업 자료 제작**: 교수/강사가 다양한 도메인의 기획 사례를 빠르게 생성하여 교육 콘텐츠 제작
