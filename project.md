# NALO(날로): 날로 먹는 프로젝트 기획

## 완전 무료 트렌드 기반 아이디어 생성 플랫폼 기획서


***

## 📋 1. 프로젝트 개요

### **서비스명**

**NALO(날로)** - *날로 먹는 프로젝트 기획*
> "노력 없이 쉽게 얻는다"는 뜻의 '날로 먹는다'에서 착안하여, **누구나 쉽고 빠르게** 트렌드 기반 프로젝트 아이디어를 얻을 수 있는 플랫폼

### **핵심 가치 제안**

- **🚀 빠르게**: 3분 내 맞춤형 아이디어 생성
- **💰 공짜로**: 모든 기능 완전 무료 이용
- **🎯 정확하게**: 실시간 트렌드 기반 개인화된 제안
- **📱 간단하게**: 복잡한 설정 없이 바로 시작


### **타겟 사용자**

- **1차 타겟**: 사이드 프로젝트를 시작하려는 개발자
- **2차 타겟**: 창업 아이템을 찾는 예비 창업자
- **3차 타겟**: 새로운 사업 기회를 모색하는 기획자


### **핵심 기능 (완전 무료)**

1. **실시간 트렌드 모니터링** - 무료 API로 글로벌 이슈 수집
2. **키워드 기반 아이디어 생성** - GPT 무료 토큰으로 맞춤 제안
3. **간편한 관심사 입력** - 선택형 + 직접입력 하이브리드
4. **실행 가능성 자동 평가** - 기술적 난이도, 시장성, 경쟁도 분석

### **Zero Cost 기술 스택**

| 구분 | 무료 서비스 | 제한사항 | 대응방안 |
| :-- | :-- | :-- | :-- |
| **AI 모델** | OpenAI GPT (정보 제공 동의) | 하루 250만 토큰 | 토큰 효율성 최적화, 캐싱 |

- **결과 캐싱**: 유사한 요청 시 캐시 활용
- **토큰 카운팅**: 실시간 사용량 모니터링


#### **웹서치 API 효율성**

- **DDGS 메인 사용**: 기본 트렌드 수집
- **Brave API 보조**: 중요한 키워드만 검증
- **RSS 피드 활용**: 뉴스 사이트 무료 피드
- **크롤링 최소화**: API 우선, 필요시에만 크롤링


#### **데이터베이스 용량 최적화**

- **텍스트 압축**: 불필요한 데이터 제거

## 💻 4. 기능 명세 (무료 최적화)

### **핵심 기능 설계**

#### **1. 실시간 트렌드 수집 (완전 무료)**

**무료 데이터 소스**

- **DDGS**: DuckDuckGo 검색 결과 (무제한)
- **Reddit API**: 인기 게시물 (일 100 요청)
- **Google Trends RSS**: 실시간 트렌드 피드
- **GitHub Trending**: 기술 트렌드
- **Product Hunt RSS**: 제품 트렌드
- **Hacker News API**: 기술 뉴스

**수집 최적화 전략**

- 시간대별 수집: 피크 타임 집중 모니터링
- 키워드 필터링: 관련성 높은 데이터만 저장
- 중복 제거: 동일 소스 데이터 정리
- 캐시 활용: 1시간 단위 업데이트


#### **2. GPT 기반 아이디어 생성 (무료 250만 토큰)**

**토큰 절약 전략**

- **압축 프롬프트**: 최소한의 정보로 최대 효과
- **한국어 최적화**: 토큰 수 적은 한국어 우선
- **배치 생성**: 한 번에 3-5개 아이디어 생성
- **템플릿 재사용**: 검증된 프롬프트 패턴 활용

**일일 토큰 배분**

- 사용자당 일평균 5,000 토큰 (아이디어 3개)
- 일 최대 500명 사용자 서비스 가능
- 피크 시간 토큰 풀링으로 효율성 극대화


#### **3. 키워드 기반 개인화**

**무료 개인화 기능**

- 로컬스토리지 활용: 사용자 선호도 저장
- 쿠키 기반 추천: 재방문시 맞춤 서비스
- 간단 회원가입: 이메일만으로 프로필 저장
- 히스토리 관리: 최근 20개 검색 기록 유지


#### **4. 실행 가능성 평가**

**자동화된 평가 시스템**

- **기술 난이도**: 키워드 기반 자동 점수
- **시장성**: 트렌드 데이터 기반 산출
- **경쟁도**: 검색 결과 수 기반 추정
- **예상 비용**: 미리 정의된 템플릿 활용
